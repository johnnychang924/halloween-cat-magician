# 墓地驚魂夜
## description:
- 操作貓咪魔法師擊退墓地中出現的鬼魂，來保護小鎮中的其他居民。
## how to play:
- 使用鍵盤輸入鬼魂頭上對應的英文字母就可以消滅鬼魂，如果被鬼魂碰到五次貓咪魔法師就會被擊敗，請加油練習英打吧!!!
